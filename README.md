# GitDapps — Web3 Wallet Platform

A production-grade Web3 crypto wallet platform built with Next.js 16, wagmi, RainbowKit, MongoDB, and Alchemy.

## Live Demo
https://gitdapps-support.vercel.app

## Features
- Connect any wallet — MetaMask, Phantom, Coinbase, WalletConnect
- Sign In With Ethereum (SIWE) — passwordless Web3 auth
- Multi-chain dashboard — Ethereum, Polygon, Arbitrum, Base, Optimism
- Real token and NFT display via Alchemy API
- Portfolio analytics with live USD values via CoinGecko
- Send ETH with live transaction tracking
- Admin panel — track all connected wallets and their assets
- Live crypto news and GitHub activity feed

## Tech Stack
Next.js 16 · TypeScript · Tailwind CSS v4 · wagmi · RainbowKit · MongoDB Atlas · iron-session · SIWE · Alchemy API · CoinGecko API · Vercel
